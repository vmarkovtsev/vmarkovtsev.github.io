Rules

  • a≤5 Λ b≤1 Λ c ⇒ α
  • a≤5 Λ 1<b<4 ⇒ β
  • 5<a<10 Λ c ⇒ γ
  • a>5 Λ c Λ b>2 ⇒ α

Rules optimization

a>5 Λ c Λ b>2 Λ d Λ a>10 ⇒ α(merge)

a>10 Λ c Λ b>2 Λ d ⇒ α(redundant)

a>10 Λ c Λ d ⇒ α(feature exclusion)

a>10 Λ c ⇒ α

Result

  • -40% ~ -60% less attributes
  • -30% ~ -50% less rules
