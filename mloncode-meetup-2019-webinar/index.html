When to help?

  • While you type = IDE

IDE

  • Instant feedback
  • Rich information
  • Rich UI
  • Instant feedback
  • Many different IDEs and languages

CI

  • Part of the workflow
  • More time to run
  • No UI
  • Must not be wrong
  • Longer feedback loop

PR reviews

  • Part of the workflow
  • More time to run
  • UI
  • Should not be wrong
  • Longer feedback loop

cron

  • Plenty of time to run
  • Outside of the workflow
  • No UI
  • Longest feedback loop
